# E 类：信息展示类模板

涵盖：信息图、概念图、要点卡片。

---

## 1 信息图 (Infographic Poster)

### 布局结构

单页纵向海报：

- 顶部：大标题 + 副标题
- 中部：3–6 个信息区块，每块"图标 + 大数字 + 简短说明"混排
- 底部：数据来源 / 日期 / 品牌标识

区块可横向并排、纵向堆叠或 Z 字型布局。

### 必填字段

- 主标题
- 3–6 个核心数据/要点
- 每要点的数字或图标
- 每要点 1 句说明

### 引导问题

想传达的核心信息是什么？
有哪些关键数据支撑？
给谁看？（内部 / 对外传播）

### HTML 布局方式

纵向 Flex 分区。每个数据区块用 Grid（图标 + 大数字 + 文字）。大数字字号 56–72px，图标用 emoji 或 SVG line icon。配色丰富但不超过 4 种主色。

---

## 2 概念图 (Concept Map)

### 布局结构

核心概念居中，周围发散若干相关概念节点。与思维导图的区别：**连线上必须带"关系动词"**（包含 / 导致 / 依赖 / 属于 / 产生），每条关系都有命名。

### 必填字段

- 核心概念
- 至少 5 个相关概念节点
- 每条连线的关系标签

### 引导问题

核心概念是什么？
相关有哪些概念？
每对概念之间是什么关系？（导致 / 包含 / 依赖 / 等价……）

### HTML 布局方式

节点用圆角矩形绝对定位或 Grid。连线用 SVG `<path>`，标签用 `<text>` 放在路径中点。不同关系类别用不同线条样式（实线=强关系，虚线=弱关系）。

---

## 3 要点卡片 (Icon Bullet Cards)

### 布局结构

Grid 3–4 列卡片墙。每张卡片：

- 顶部大图标（emoji 或 SVG，48–64px）
- 加粗标题（1 行，≤10 字）
- 说明文字（2–3 行，≤40 字）

适用于：产品卖点罗列、方法论要点、Feature List。

### 必填字段

- 至少 3 张卡片
- 每张：图标 + 标题 + 说明
- 可选：强调标签（"新"、"推荐"）

### 引导问题

要罗列多少个要点？
每个要点的核心关键词？
哪几个需要突出强调？

### HTML 布局方式

Grid auto-fit 自动排列卡片。卡片用圆角 border + 轻阴影 + hover 微抬起（尽管静态截图）。图标用色块背景（圆形或圆角方形）衬托。标题用 title 字体，说明用 body 字体。
