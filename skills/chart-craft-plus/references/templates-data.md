# D 类：数据可视化类模板

涵盖：ER 图、漏斗图、桑基图、热力矩阵、时间轴、韦恩图、对比表、KPI 看板。

---

## 1 ER 图 (ER Diagram)

### 布局结构

实体用卡片：表头（实体名）+ 字段列表（PK/FK 带标注）。关系线连接实体，两端标基数（1:1、1:N、N:M）。

### 必填字段

- 至少 2 个实体
- 每实体字段及主键标识
- 关系类型

### 引导问题

系统有哪些核心实体？每个有什么字段？它们怎么关联？

### HTML 布局方式

Grid 或绝对定位实体卡。关系用 SVG `<line>` + 两端文字基数。PK 用钥匙 emoji 或 "PK" 标签前缀，FK 同理。

---

## 2 漏斗图 (Funnel)

### 布局结构

纵向从上到下，每层一个梯形/倒三角条，宽度表示数量。右侧标百分比/转化率。支持 AARRR（获取/激活/留存/收益/推荐）模板。

### 必填字段

- 至少 3 层
- 每层名称 + 数量
- 层间转化率

### 引导问题

漏斗有几级？每级的绝对值？关心哪个环节的流失？

### HTML 布局方式

纵向 Flexbox 堆叠。每层用 clip-path 裁成梯形，宽度按比例设置。右侧绝对定位百分比标签。

---

## 3 桑基图 (Sankey)

### 布局结构

左右两端分别是源节点和目标节点，中间用加粗贝塞尔曲线流动条连接，条宽表示流量大小。多级可多列排布。

### 必填字段

- 至少 2 个来源
- 至少 2 个去向
- 每条流量值

### 引导问题

流量/预算/用户 从哪里来？流向哪里？各路径占比？

### HTML 布局方式

纯 SVG：左右两列节点矩形，中间用 `<path d="M...C...">` 贝塞尔曲线，stroke-width 按流量设置，半透明填充。**必须附图例**说明颜色分类。

---

## 4 热力矩阵 (Heatmap)

### 布局结构

二维网格，行列标签，每格填色表示强度（浅→深）。右侧或底部图例色带。

### 必填字段

- 行维度和标签
- 列维度和标签
- 每格数值
- 数值范围

### 引导问题

两个维度是什么？每格数值从哪来？关心低值还是高值？

### HTML 布局方式

CSS Grid。每格 background-color 按数值插值（白→目标色）。格子内可叠加数字。**必须附色阶图例**。

---

## 5 时间轴 (Timeline)

### 布局结构

横向主线，事件节点按时间先后排布在主线上下两侧交替。每节点：日期 + 标题 + 简述。

### 必填字段

- 至少 4 个事件
- 每事件日期 + 标题
- 简述（≤20 字）

### 引导问题

从什么时候开始？有哪些关键节点？每个节点发生了什么？

### HTML 布局方式

横向 Flexbox 主线（1px 实线）。事件用绝对定位 + 小圆点连接主线。文字卡片上下交替避免重叠。

---

## 6 韦恩图 (Venn)

### 布局结构

2–4 个半透明圆形相交。交集区域写共同特征，独占区域写独有特征。中心三重交集写核心。

### 必填字段

- 2–4 个集合名
- 每集合独有特征
- 两两交集 + 全交集内容

### 引导问题

有几个集合？各自独有什么？交集的共性是什么？

### HTML 布局方式

SVG 绘制半透明 `<circle>`（opacity 0.4），混合叠加产生交集色。文字用绝对定位覆盖在对应区域。

---

## 7 对比表 (Comparison Table)

### 布局结构

行 = 对比维度，列 = 选项/产品。单元格可用 ✓/✗、星级、金额、短描述、色块等表达。表头列可加图标 + 名称 + 副标题。

### 必填字段

- 至少 2 个选项
- 至少 4 个维度
- 每格明确结论

### 引导问题

对比哪些选项？从哪些维度？哪些是硬指标哪些是主观评分？

### HTML 布局方式

纯 HTML table 或 Grid。推荐列加高亮边框 + "推荐"角标。✓ 用绿色、✗ 用灰色/红色。

---

## 8 KPI 看板 (KPI Dashboard)

### 布局结构

顶部摘要卡片行（4–6 个核心指标：值 + 同比 + 迷你趋势线）。下方分区域的图表（折线趋势、饼图分布、排行榜）组合。

### 必填字段

- 至少 4 个核心 KPI
- 每 KPI 当前值、目标值、同比变化
- 至少 1 个趋势数据

### 引导问题

核心 KPI 有哪些？每个当前值和目标？时间范围？

### HTML 布局方式

顶部 Grid 横排指标卡（大数字 + 上下箭头同比 + SVG 迷你 sparkline）。下方 Grid 2–3 列放图表组件。配色：正向指标绿、负向红、中性 slate。
